How it works

The Technique

These underline effects use CSS pseudo-elements (::after or ::before) with width transitions to create smooth hover animations. The underline is positioned absolutely below the link text and animated from 0 to 100% width on hover.

Key Properties

  • position: absolute - Position the underline relative to the link
  • bottom: 0 - Align underline to the bottom of the text
  • width: 0 → 100% - Expand the underline on hover
  • transition - Create smooth animation
