# Holy Grail Layout

A classic CSS layout pattern that creates a responsive page structure with header, footer, and three-column content area.

## Features

- Header and footer span full width
- Three-column layout with sidebars
- Main content expands to fill available space
- Responsive: stacks vertically on mobile
- Footer stays at bottom regardless of content length

## Use Cases

- Blog layouts with sidebar navigation
- Admin dashboards with side panels
- Documentation sites with table of contents
- E-commerce pages with filters and recommendations

## Browser Support

Works in all modern browsers that support Flexbox (IE11+, all evergreen browsers).

## Demo

Open `index.html` to see the interactive demonstration with controls to toggle sidebars and add dynamic content.
