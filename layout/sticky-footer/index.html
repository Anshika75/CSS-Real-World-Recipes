Sticky Footer Layout

The footer stays at the bottom of the page, even with minimal content

Main Content Area

This content area can grow as needed. The footer will always stick to the bottom of the viewport when content is short, or appear after all content when the page is long.

Try resizing your browser window to see how it works!

The CSS Technique

.page-container {
  min-height: 100vh;
  display: flex;
  flex-direction: column;
}

.main-content {
  flex: 1;  /* Grows to fill space */
}

.footer {
  margin-top: auto;  /* Pushes to bottom */
}

Click to see how the footer behaves with different content heights

Additional Content

When you have more content, the footer naturally moves down and appears after all the content. This is the beauty of the flexbox sticky footer technique!
