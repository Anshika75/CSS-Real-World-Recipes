The CSS

.grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  gap: 24px;
}

How it works: The auto-fit keyword automatically fits as many columns as possible, while minmax(250px, 1fr) ensures each item is at least 250px wide but can grow to fill available space. Try auto-fill to see the difference - it keeps ghost columns!

auto-fit vs auto-fill

auto-fit

Collapses empty tracks and stretches items to fill the available space. Best for fluid layouts where you want items to grow.

auto-fill

Keeps ghost columns even if empty, preventing items from stretching too wide. Best when you want to maintain consistent sizing.
