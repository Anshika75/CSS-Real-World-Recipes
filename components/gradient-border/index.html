Gradient Border Cards

Beautiful, responsive gradient borders using CSS masking

Primary Gradient

This card uses the signature pink-to-red gradient for its border effect.

Default

Cyan Accent

A cool teal to cyan transition that contrasts well with the primary brand colors.

Accent

Warm Amber

Warm yellow to orange tones create a high-visibility border for special content.

Warning

How it works

.card::before {
  content: "";
  position: absolute;
  inset: 0;
  padding: 2px; /* Border thickness */
  background: linear-gradient(135deg, pink, red);
  -webkit-mask: 
    linear-gradient(#fff 0 0) content-box, 
    linear-gradient(#fff 0 0);
  mask-composite: exclude;
}
